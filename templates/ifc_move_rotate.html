IFC Move / Rotate

WIP · Whole-model transform

Reposition and rotate an entire IFC model when shared coordinates are wrong. This applies one global transform: T = T2 * Rz * T1 where T1 translates the current point to origin, Rz rotates about global Z, and T2 translates to the target point.

Warning: This tool transforms model placements. If the IFC already contains valid georeferencing / map conversion data, verify whether that metadata also needs updating outside this tool.
